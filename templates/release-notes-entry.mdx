## First Release

This is the initial release of the BYU-Idaho Support Agent. It is a beta release available to students, employees, and visitors to BYU-Idaho for a limited time. This release features the chat and voice chat interfaces, a visually appealing home page, and a comprehensive documentation website.

This release was a collaborative effort between the BYU-Idaho Information Technology team and the BYU-Idaho Support Center. The Support Center provided feedback and testing for this release.
